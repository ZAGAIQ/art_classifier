1. Введение
Настоящий отчёт описывает результаты выполнения курсового проекта по дисциплине «Основы Проектной Работы». Проект реализован командой из 6 человек в рамках Варианта 1 — «Импрессионизм: анализ светотени и цветовых переходов».

1.1. Цель проекта
Разработать веб-приложение АртВизор, которое принимает изображение от пользователя, определяет вероятность принадлежности к импрессионизму и возвращает список наиболее схожих произведений из базы данных с указанием процента сходства.

1.2. Краткое описание решения
Приложение реализовано на основе следующего технического стека:
CLIP (openai/clip-vit-base-patch32) — извлечение 512-мерных эмбеддингов из изображений
Keras / TensorFlow — бинарный классификатор (импрессионизм / не-импрессионизм)
Flask — REST API с эндпоинтом POST /analyze
HTML/CSS/JavaScript — пользовательский интерфейс (artvizor_mvp.html)
Датасет — 200+ изображений 6 художественных стилей

2. Product Owner — Мальцев Тимофей
2.1. Визуальные признаки импрессионизма
Product Owner сформировал чек-лист из 25 визуальных признаков импрессионизма, сгруппированных по 5 категориям, которые используются как критерии для оценки изображений и обучения модели.

Категория 1: Техника мазка
Видимые короткие мазки — фактура поверхности различима при увеличении (вес: 5)
Пастозная техника — краска нанесена толстым слоем, есть рельеф (вес: 4)
Динамичные мазки — следуют форме объекта или движению (вес: 4)
Нет сглаживания — отсутствует академическая полировка (вес: 5)
Оптическое смешение цветов — пуантилистический эффект на расстоянии (вес: 3)

Категория 2: Цветовая палитра
Нет чёрного цвета — тени написаны синим/фиолетовым (вес: 5)
Яркие насыщенные цвета — чистые спектральные оттенки (вес: 4)
Рефлексы среды — объекты отражают цвет неба/воды/листвы (вес: 4)
Пастельные переходы — мягкие тональные переходы без резких контуров (вес: 3)
Тёплый/холодный контраст — чередование тёплых и холодных зон (вес: 4)
Белёсая дымка (валёр) — общая светлота выше нормы (вес: 3)

Категория 3: Светотень
Пленэрное освещение — рассеянный свет, мягкие тени (вес: 5)
Мерцающий свет — ощущение движения, живой эффект (вес: 4)
Цветные тени — синие/фиолетовые тени вместо серых (вес: 5)
Нет жёсткого блика — отсутствует академическая лепка формы (вес: 3)
Конкретное время суток/погода — передача момента освещения (вес: 4)

Категория 4: Композиция
Эскизность — периферия менее детализирована (вес: 3)
Воздушная перспектива — дальние объекты теряют чёткость (вес: 4)
Срезанные фигуры — часть объекта выходит за рамку (вес: 3)
Акцент на моменте — нет нарратива, запечатлён миг (вес: 3)

Категория 5: Тематика
Природа и вода — реки, пруды, море, поля, сады (вес: 3)
Городская жизнь Парижа — бульвары, кафе, парки (вес: 3)
Досуг и развлечения — балы, пикники, танцы, лодки (вес: 3)
Балет и театр — танцовщицы, музыканты, зрители (вес: 4)
Натюрморт с цветами — букеты и вазы (вес: 2)

2.2. Пользовательские сценарии
Сформированы два основных сценария использования приложения АртВизор:

Сценарий 1 — Пользователь хочет узнать стиль картины
Пользователь открывает artvizor_mvp.html в браузере
Нажимает кнопку «Выбрать изображение» и загружает файл JPEG/PNG
Нажимает «Анализировать»
Получает вероятность импрессионизма в процентах
Если вероятность ≥ 50% — видит топ-5 похожих произведений с указанием автора, названия и % сходства

Сценарий 2 — Пользователь исследует датасет
Пользователь загружает фотографию пейзажа или натюрморта
Приложение определяет, есть ли в изображении характерные признаки импрессионизма
При обнаружении — приложение предлагает визуально похожие работы из базы данных
Пользователь сравнивает загруженное изображение с предложенными произведениями

4. Data Engineer / ML Engineer — Загородников Дмитрий
4.1. Сбор и подготовка датасета
Для обучения модели был собран сбалансированный датасет из 200+ изображений, разделённых на 6 классов художественных стилей:
Impressionism — импрессионизм (основной класс)
Post_Impressionism — пост-импрессионизм
Baroque — барокко
Cubism — кубизм
Minimalism — минимализм
Ukiyo_e — укиё-э (японская гравюра)

Источники изображений: Google Arts & Culture, WikiArt, Roboflow. Изображения размечены вручную и сохранены в структуре папок balanced_art_dataset/.

4.2. Извлечение CLIP-эмбеддингов
Для каждого изображения была запущена CLIP-модель (openai/clip-vit-base-patch32), которая извлекает 512-мерный вектор признаков. Все эмбеддинги L2-нормализуются и сохраняются в файл embeddings.npz.


4.3. Обучение классификатора
На основе CLIP-эмбеддингов обучена лёгкая нейросеть Keras для бинарной классификации: импрессионизм (1) / не-импрессионизм (0).

6. Fullstack Developer — Ковнер Степан
6.1. Flask Backend (app.py)
Разработан Flask-сервер, реализующий REST API для анализа изображений. Сервер использует ленивую загрузку моделей при первом запросе.

Эндпоинты API

Формат ответа POST /analyze
probability — вероятность импрессионизма в % (0..100)
is_impressionist — true если probability ≥ threshold
threshold — порог классификации (50.0%)
similar[] — топ-5 похожих работ (только если is_impressionist: true)
  → artist, title, similarity (%), filename

Алгоритм работы /analyze
Принять изображение (multipart/form-data, поле image)
Открыть через Pillow, конвертировать в RGB
Извлечь CLIP-эмбеддинг → L2-нормализация
Передать в Keras-классификатор → вероятность (0..1)
Если ≥ 0.5: вычислить косинусное сходство np.dot() с базой → топ-5
Вернуть JSON-ответ

6.2. HTML/JS Фронтенд (artvizor_mvp.html)
Разработан одностраничный интерфейс без фреймворков на HTML5/CSS3/JavaScript. Фронтенд отправляет запросы к Flask API через fetch() и отображает результаты.

Загрузка изображения через <input type=file>
Клиентская компрессия изображений через Canvas API (для файлов > 5 МБ)
Отправка POST /analyze с изображением
Отображение вероятности и списка похожих работ
Flask-CORS разрешает запросы с file:// и localhost

6.3. Известные ограничения
Холодный старт: первый запрос занимает 30–80 сек (загрузка CLIP ~400 МБ)
CPU-режим: без CUDA инференс занимает 5–10 сек на изображение
Поддерживаемые форматы: JPEG, PNG (файлы > 8 МБ отклоняются с кодом 413)
