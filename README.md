# АртВизор: Интеллектуальный классификатор произведений импрессионизма

**АртВизор (ArtVizor)** — интеллектуальная веб-система для анализа и классификации произведений искусства. Система определяет, относится ли картина к стилю **импрессионизм**, на основе визуальных признаков, извлеченных с помощью мультимодальной нейросети **CLIP** от OpenAI, и рекомендует визуально схожие произведения из локальной базы данных.

В проект входят: скрипты подготовки данных и обучения модели, консольный анализатор, а также полноценный веб-прототип с интерактивным интерфейсом.

---

## 🏛️ Архитектура системы

Система состоит из трёх основных компонентов:
1. **Экстрактор признаков (Feature Extractor)**: Предобученная нейросеть **CLIP** (`openai/clip-vit-base-patch32`), которая преобразует изображение в 512-мерный вектор (эмбеддинг). Полученные векторы нормализуются по L2-норме.
2. **Классификатор (Classifier)**: Легковесная модель **Keras** (фактически логистическая регрессия в виде одного слоя `Dense(1, activation="sigmoid")` с L2-регуляризацией), обучающаяся разделять стили на основе эмбеддингов. Содержит всего **513 обучаемых параметров**, что исключает переобучение и обеспечивает отличную обобщающую способность.
3. **Рекомендательная система**: Сравнивает нормализованный эмбеддинг загруженного изображения с эмбеддингами импрессионистских картин из базы данных методом косинусного сходства (скалярное произведение векторов) и выводит наиболее близкие по стилю и композиции работы.

## ⚡ Быстрый запуск

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python (рекомендуется 3.10 или 3.11).

```bash
# Создание виртуального окружения
python -m venv .venv

# Активация окружения (Windows)
.venv\Scripts\activate

# Активация окружения (Linux/macOS)
source .venv/bin/activate

# Установка зависимостей (включает Flask, Keras, PyTorch, Transformers, Flask-CORS)
pip install -r requirements.txt
```

### 2. Извлечение эмбеддингов
Перед обучением классификатора необходимо извлечь CLIP-признаки из папки датасета `balanced_art_dataset`.

```bash
python extract_embeddings.py
```
Скрипт создаст сжатый архив `embeddings.npz`, который будет служить базой данных для обучения классификатора и рекомендаций.

### 3. Обучение модели
Запустите скрипт обучения для создания файла модели и оценки её качества:

```bash
python train_classifier.py
```
Модель обучается до **100 эпох** с механизмом ранней остановки (Early Stopping). После завершения в корне проекта появятся файлы:
* `impressionism_classifier.keras` — сохраненная модель.
* `metrics.json` — подробный лог метрик на 985 валидационных изображениях.
* Графики обучения `learning_curves.png`, `confusion_matrix.png` и `roc_curve.png`.

### 4. Запуск веб-прототипа (Интерфейс)
Запустите веб-сервер Flask:

```bash
python app.py
```
Откройте в браузере страницу **[http://127.0.0.1:5000/](http://127.0.0.1:5000/)**. Перетащите туда картину, нажмите «Анализировать», и сайт покажет вероятность стиля, статус классификации и миниатюры визуально схожих картин из базы данных.

### 5. Консольный анализ картин
Вы также можете классифицировать изображение прямо из терминала:

```bash
python analyze_image.py testimg/imagei1.png --threshold 0.5 --top_n 5
```

## 🔌 Спецификация REST API

Сервер Flask поддерживает CORS (запросы разрешены с любых источников) и ограничивает размер загружаемых файлов до **8 МБ** (файлы большего размера отклоняются с HTTP-кодом `413 Payload Too Large`).

### 1. Внутренний эндпоинт фронтенда (`POST /api/analyze`)
Используется интерактивным интерфейсом `templates/index.html`.

* **Тело запроса (multipart/form-data):**
  * `image`: файл изображения (обязательный)
  * `threshold`: порог классификации (опционально, по умолчанию `0.5`, шкала `0.0..1.0`)
  * `top_n`: количество рекомендаций (опционально, по умолчанию `6`)
* **Формат JSON-ответа:**
  ```json
  {
    "is_impressionism": true,
    "probability": 0.8984771370887756,
    "similar_images": [
      {
        "artist": "Claude Monet",
        "title": "Water Lilies",
        "similarity": 92.18,
        "filename": "Impressionism/claude-monet_water-lilies.jpg"
      }
    ]
  }
  ```

### 2. Официальный эндпоинт по Отчёту ОПР (`POST /analyze`)
Полностью соответствует спецификации научной работы ОПР.

* **Тело запроса (multipart/form-data):**
  * `image`: файл изображения (обязательный)
  * `threshold`: порог классификации (опционально, по умолчанию `0.5`, шкала `0.0..1.0`)
  * `top_n`: количество рекомендаций (опционально, по умолчанию `5`)
* **Формат JSON-ответа:**
  ```json
  {
    "is_impressionist": true,
    "probability": 89.85,
    "threshold": 50.0,
    "similar": [
      {
        "artist": "Claude Monet",
        "title": "Water Lilies",
        "similarity": 92.18,
        "filename": "Impressionism/claude-monet_water-lilies.jpg"
      }
    ]
  }
  ```

## 📊 Метрики качества модели

Оценка обученного линейного классификатора проводилась на репрезентативной валидационной выборке из **985 изображений** (20% от общего сбалансированного датасета размером **4922 изображения**):

* **Accuracy (Точность классификации)**: ~89.85%
* **Precision (Точность)**: ~87.84%
* **Recall (Полнота)**: ~92.18%
* **F1-Score**: ~89.96%
* **ROC-AUC**: ~0.9552

Обучающая точность (на 3937 изображениях) составляет **90.52%**, что почти равно валидационной точности (**89.85%**). Это доказывает отсутствие переобучения (overfitting) благодаря оптимально подобранной архитектуре модели.
