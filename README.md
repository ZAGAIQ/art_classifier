# Классификатор произведений импрессионизма (Art Classifier)

Данный проект представляет собой интеллектуальную систему для классификации живописи. Он определяет, относится ли картина к стилю **импрессионизм**, на основе визуальных признаков, извлеченных с помощью мультимодальной нейросети **CLIP** от OpenAI, и подбирает визуально схожие произведения из локальной базы данных.

В проект входят: скрипты подготовки данных и обучения модели, консольный анализатор, а также полноценный веб-прототип с интерактивным интерфейсом.

---

## 🏛️ Архитектура системы

Система состоит из трёх основных компонентов:
1. **Экстрактор признаков (Feature Extractor)**: Замораживаемая нейросеть **CLIP** (`openai/clip-vit-base-patch32`), которая преобразует изображение в 512-мерный вектор (эмбеддинг). Векторы нормализуются по L2-норме.
2. **Классификатор (Classifier)**: Легковесная модель **Keras** (фактически логистическая регрессия в виде одного слоя `Dense(1, activation="sigmoid")` с L2-регуляризацией), обучающаяся разделять стили на основе эмбеддингов. Всего содержит **513 обучаемых параметров**, что исключает переобучение и обеспечивает отличную обобщающую способность.
3. **Рекомендательная система**: Сравнивает нормализованный эмбеддинг загруженного изображения с эмбеддингами импрессионистских картин из базы данных методом косинусного сходства (скалярное произведение векторов) и выводит наиболее близкие по стилю и композиции работы.

## ⚡ Быстрый запуск

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python (рекомендуется 3.10 или 3.11).

```bash
# Создание виртуального окружения
python -m venv .venv

# Активация окружения (Windows)
.venv\Scripts\activate

# Активация окружения (Linux/macOS)
source .venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

*Примечание: Если вы находитесь за прокси-сервером и pip выдает ошибку SOCKS-соединения, установите PySocks:*
```bash
pip install pysocks
```

### 2. Извлечение эмбеддингов
Перед обучением классификатора необходимо извлечь CLIP-признаки из папки датасета `balanced_art_dataset`.

```bash
python extract_embeddings.py
```
Скрипт создаст сжатый архив `embeddings.npz`, который будет служить базой данных для обучения классификатора и рекомендаций.

### 3. Обучение модели
Запустите скрипт обучения для создания файла модели и оценки её качества:

```bash
python train_classifier.py
```
После завершения в корне проекта появятся файлы:
* `impressionism_classifier.keras` — сохраненная модель.
* `metrics.json` — подробный лог метрик.
* Графики обучения `learning_curves.png`, `confusion_matrix.png` и `roc_curve.png`.

### 4. Запуск веб-прототипа (Интерфейс)
Запустите веб-сервер Flask:

```bash
python app.py
```
Откройте в браузере страницу **[http://127.0.0.1:5000/](http://127.0.0.1:5000/)**. Вы увидите интерактивное окно: перетащите туда картину, нажмите «Анализировать», и сайт покажет вероятность стиля, статус классификации и миниатюры визуально схожих картин из базы данных.

### 5. Консольный анализ картин
Вы также можете классифицировать изображение прямо из терминала:

```bash
python analyze_image.py testimg/imagei1.png --threshold 0.5 --top_n 5
```

## 📊 Метрики качества модели

Обученный линейный классификатор демонстрирует высокую обобщающую способность на валидационном наборе данных:

* **Accuracy (Точность классификации)**: ~89.85%
* **Precision (Точность)**: ~87.84%
* **Recall (Полнота)**: ~92.18%
* **F1-Score**: ~89.96%
* **ROC-AUC**: ~0.9552

Обучающая точность составляет **90.52%**, что почти равно валидационной точности (**89.85%**). Это доказывает отсутствие переэффекта переобучения (overfitting) благодаря оптимально подобранной архитектуре модели.
